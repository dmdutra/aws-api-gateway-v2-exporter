## Installation

```bash
python setup.py install
```

## Usage

```bash
Environment variables

Usage:
    export REFRESH_INTERVAL=20
    export PORT=9095
    export REGION=sa-east-1
    export API_ID=610lzlx1
    export API_STAGE="\$default"

Command:
    prometheus-api-gateway-http-exporter
```

<hr>

## Running the exporter

```bash
prometheus-api-gateway-http-exporter
```
